# Stock Price Tracker

A simple Python tool to fetch stock price data and plot the closing price for the last month.

## Features
- Fetches 1-month daily price data
- Plots closing prices using Matplotlib
- Works with common tickers (AAPL, MSFT, TSLA, etc.)

## Requirements
- Python 3.x

## Installation
```bash
pip install yfinance matplotlib
python stock_tracker.py
